Livret scolaire électronique pour La Grande Classe : un espace sécurisé qui consolide le suivi des étudiants, leurs livrables et les événements pédagogiques associés. Le dashboard offre une vue synthétique (progression, jalons clés, documents) pour fluidifier la communication entre l'équipe pédagogique et les apprenants.

## Lancer le projet

Installer les dépendances :

```bash
npm install
```

Démarrer le serveur Next.js en mode développement :

```bash
npm run dev
```

Construire et lancer le bundle de production :

```bash
npm run build
npm start
```
